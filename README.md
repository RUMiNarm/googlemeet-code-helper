# GoogleMeetCode Auto Input

GoogleMeet でコードやニックネームを自動で入力してくれます。

アイコンは暫定です。今後変更予定です。

# Features 執筆中

コード入力欄の下のコードボタンをクリックすることで入力と参加が同時に行われます。

Auto join を OFF にすると参加されずに入力だけ実行されます。
